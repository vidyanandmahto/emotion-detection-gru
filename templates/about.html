The project, in plain terms

How EmotionLens reads a sentence

A recurrent neural network classifies free text into six emotions: joy, sadness, anger, fear, love and surprise.

Pipeline

  1. Cleaning — lowercasing, URL/number/punctuation stripping, stopword removal, lemmatization.
  2. Tokenization — a 12,000-word Keras vocabulary turns text into integer sequences, padded to a fixed length.
  3. Modeling — SimpleRNN, LSTM, and GRU networks were each trained with an embedding layer, dropout regularization, and a softmax output; the best performer on held-out test data was kept.
  4. Serving — the saved Keras model, tokenizer, and label encoder power this Flask app in real time.

Dataset

Trained on the open Emotion dataset — roughly 20,000 short English sentences, each labeled with one of six emotions, split into train, validation, and test sets.

Tech Stack

Python TensorFlow / Keras scikit-learn NLTK Flask HTML / CSS / JS
